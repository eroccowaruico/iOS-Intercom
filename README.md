# Intercom iOS App

## Xcode Quick Start

1. Open `IntercomApp.xcodeproj` in Xcode.
2. Select the `IntercomApp` scheme.
3. Choose an iOS Simulator (e.g. iPhone 15).
4. Build & Run.

## Notes

- The app target includes the core transport/VAD sources under `Sources/IntercomCore`.
- The default AppIcon asset catalog is included so the project builds without additional assets.
