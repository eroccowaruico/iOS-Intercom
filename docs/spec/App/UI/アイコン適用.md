# RideIntercom UI アイコン適用

## 目的

本書は `docs/design system/アイコン定義.md` を RideIntercom App の UI 項目へ適用する正本である。

## Tabs / Group Selection

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `callTab` | 通話タブ | `waveform.circle.fill` | 通話機能入口 |
| `diagnosticsTab` | 診断タブ | `gauge` | 計測・観測入口 |
| `settingsTab` | 設定タブ | `gearshape.fill` | 設定入口 |
| `groupRow-{group.name}` | グループ行先頭 | `person.3` | 接続中は色で補助 |
| `groupRow-{group.name}` | 行末遷移示唆 | `chevron.right` | 行末専用 |
| `createGroupButton` | グループ作成 | `plus` | 新規作成 |
| なし | グループ削除 | `trash` | swipe / context menu の破壊操作 |

## Call

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `showGroupsButton` | グループ一覧戻り | `person.3.fill` | 一覧への復帰導線 |
| `connectionStatusIcon` | 接続状態 | `wifi.slash` / `wifi.exclamationmark` / `wifi` / `exclamationmark.triangle.fill` | 接続成否の最終状態 |
| `routeLabel` | 通信方式補助 | `dot.radiowaves.left.and.right` / `network` | Local / Internet |
| `duckingStatusIcon` | ダッキング状態補助 | `waveform` | 設定 ON 時のみ |
| `localMicrophoneHeaderControl` / `localMicrophoneMuteButton` | ローカル入力 | `mic.fill` / `mic.slash.fill` | Live / Muted |
| `outputStateLabel` / `masterOutputMuteButton` | マスター出力 | `speaker.wave.2.fill` / `speaker.slash.fill` | 出力状態 |
| `connectButton` / `disconnectButton` | 接続開始 / 切断 | `antenna.radiowaves.left.and.right` / `xmark.circle.fill` | 操作アイコン |
| `inviteButton` | 招待共有 | `square.and.arrow.up` | 共有操作 |
| `participantStatusSummary{index}` | 参加者接続 / 認証 | `wifi` 系 / `checkmark.seal.fill` 系 | 接続軸と認証軸を分離 |
| `participantAudioPipelineState{index}` | codec 表示 | `cpu` | 音声処理補助 |

## Diagnostics

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `realDeviceCallDebugSummaryLabel` | 通話/経路 | `checklist` | 通話全体 |
| `pipeline-session-step` | 音声セッション | `waveform` | SessionManager report |
| `pipeline-input-step` | 入力stream | `mic.fill` / `mic.slash.fill` | capture と voice processing |
| `receive-pipeline-output-step` | 出力stream | `speaker.wave.2.fill` / `speaker.slash.fill` | renderer と schedule |
| `pipeline-codec-step` / `receive-pipeline-codec-step` | codec | `cpu` | requested / selected / fallback |
| `receptionDebugSummaryLabel` | route metrics | `clock.arrow.circlepath` | RTT / jitter / loss |
| `pipeline-mixer-step` / `receive-pipeline-peer-bus-step` | mixer | `slider.horizontal.3` / `person.2.wave.2` | bus / output / volume |
| `receive-pipeline-mix-step` | mix down | `arrow.triangle.merge` | peer bus から RX master への集約 |
| `receive-pipeline-master-bus-step` | RX master | `waveform.path.ecg` | master bus / output volume |
| `pipeline-effects-step` / `receive-pipeline-master-effects-step` | effect chain | `wand.and.sparkles` | AudioMixer effect chain |
| `inviteDebugSummaryLabel` | 招待 | `square.and.arrow.up` | invite 状態 |
