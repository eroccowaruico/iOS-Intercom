# RideIntercom UI アイコン適用

## 目的

本書は `docs/design system/アイコン定義.md` を RideIntercom App の UI 項目へ適用する正本である。

## Tabs / Group Selection

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `tab-call` | 通話タブ | `waveform.circle.fill` | 通話機能入口 |
| `tab-diagnostics` | 診断タブ | `gauge` | 計測・観測入口 |
| `tab-settings` | 設定タブ | `gearshape.fill` | 設定入口 |
| `group-row-icon` | グループ行先頭 | `person.3` | 接続中は色で補助 |
| `group-row-disclosure` | 行末遷移示唆 | `chevron.right` | 行末専用 |
| `create-group-button` | グループ作成 | `plus` | 新規作成 |
| `group-row-swipe-delete` / `group-row-context-delete` | グループ削除 | `trash` | 破壊操作 |

## Call

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `show-groups-button` | グループ一覧戻り | `person.3.fill` | 一覧への復帰導線 |
| `call-connection-icon` | 接続状態 | `wifi.slash` / `wifi.exclamationmark` / `wifi` / `exclamationmark.triangle.fill` | 接続成否の最終状態 |
| `call-route-label` | 通信方式補助 | `dot.radiowaves.left.and.right` / `network` | Local / Internet |
| `call-ducking-status-icon` | ダッキング状態補助 | `waveform` | 設定 ON 時のみ |
| `local-microphone-header` / `local-microphone-mute-button` | ローカル入力 | `mic.fill` / `mic.slash.fill` | Live / Muted |
| `master-output-group` / `master-output-mute-button` | マスター出力 | `speaker.wave.2.fill` / `speaker.slash.fill` | 出力状態 |
| `connect-disconnect-button` | 接続開始 / 切断 | `antenna.radiowaves.left.and.right` / `xmark.circle.fill` | 操作アイコン |
| `invite-button` | 招待共有 | `square.and.arrow.up` | 共有操作 |
| `participant-status-icons` | 参加者接続 / 認証 | `wifi` 系 / `checkmark.seal.fill` 系 | 接続軸と認証軸を分離 |
| `participant-codec-label` | codec 表示 | `cpu` | 音声処理補助 |

## Diagnostics

| 項目ID | 項目 | 仕様アイコン | 補足 |
|---|---|---|---|
| `diag-call-route-summary` | 通話/経路 | `checklist` | 通話全体 |
| `diag-session-summary` | 音声セッション | `waveform` | SessionManager report |
| `diag-input-stream-summary` | 入力stream | `mic.fill` / `mic.slash.fill` | capture と voice processing |
| `diag-output-stream-summary` | 出力stream | `speaker.wave.2.fill` / `speaker.slash.fill` | renderer と schedule |
| `diag-codec-summary` | codec | `cpu` | requested / selected / fallback |
| `diag-route-metrics-summary` | route metrics | `clock.arrow.circlepath` | RTT / jitter / drop |
| `diag-mixer-summary` | mixer | `waveform.path.ecg` | bus / output / volume |
| `diag-auth-summary` | 認証 | `checkmark.seal.fill` | authenticated peer |
| `diag-invite-summary` | 招待 | `square.and.arrow.up` | invite 状態 |
