# RideIntercom 設定値一覧

## 方針

本ドキュメントは、現行実装で利用する設定値を「ユーザー変更可否」と「反映タイミング」で整理する。  
画面上で設定可能な値は、タブ/セクション/コントロールまで場所を明記する。

## 設定値カタログ

| 設定名 | キー/識別 | 型 | 既定値 | 範囲/候補 | 反映タイミング | ユーザー変更 | 設定場所 |
|---|---|---|---|---|---|---|---|
| 入力デバイス | selectedInputPort | AudioPortInfo | Auto | 端末の利用可能入力ポート | Live（Audio session有効時）/Next start（未有効時） | 可 | Settingsタブ > Audio I/O > Input Picker |
| 出力デバイス | selectedOutputPort | AudioPortInfo | Auto | 端末の利用可能出力ポート | Live（Audio session有効時）/Next start（未有効時） | 可 | Settingsタブ > Audio I/O > Output Picker |
| 送信codec | preferredTransmitCodec | AudioCodecIdentifier | pcm16 | pcm16, heAACv2 | 即時（次packet生成から） | 可 | Settingsタブ > Transmit Codec > Codec Segmented Picker |
| HE-AAC品質 | heAACv2Quality | HEAACv2Quality | Medium | Low(16kbps), Medium(24kbps), High(40kbps) | 即時（次encoder session更新から） | 可 | Settingsタブ > Transmit Codec > HE-AAC v2 Quality Segmented Picker（codec=HE-AAC時のみ表示） |
| VAD閾値 | voiceActivityDetectionThreshold | Float | 0.003 | 0.00025〜0.12（step 0.00025） | 即時 | 可 | Settingsタブ > Audio Check > Voice Activity Detection Threshold Slider |
| Sound Isolation | isSoundIsolationEnabled | Bool | 起動時の実装既定（対応時） | ON/OFF | 即時 | 可（対応デバイスのみ） | Settingsタブ > Audio Check > Sound Isolation Toggle |
| Audio Check実行 | audioCheckPhase / startAudioCheck | Enum / Action | idle | idle, recording, playing, completed, failed | ボタン実行時 | 可（実行操作） | Settingsタブ > Audio Check > Record 5s and Play Button |
| マスター出力音量 | masterOutputVolume | Float | 1.0 | 0.0〜1.0 | 即時 | 可 | Callタブ > 通話画面ヘッダー > Output Slider |
| マスター出力ミュート | isOutputMuted | Bool | false | ON/OFF | 即時 | 可 | Callタブ > 通話画面ヘッダー > Output Mute Button |
| 参加者別出力音量 | remoteOutputVolumes[peerID] | Float | 1.0（未設定時） | 0.0〜1.0 | 即時 | 可 | Callタブ > Participants > 各参加者カード > Output Slider |
| ローカルマイクミュート | isMuted | Bool | false | ON/OFF | 即時 | 可 | Callタブ > 通話画面ヘッダー > Local microphone mute control |

## 内部固定設定（画面変更不可）

| 設定名 | キー/識別 | 型 | 設定値 | 用途 |
|---|---|---|---|---|
| VAD最小閾値 | VoiceActivityDetector.minThreshold | Float | 0.00025 | VAD下限 |
| VAD最大閾値 | VoiceActivityDetector.maxThreshold | Float | 0.12 | VAD上限 |
| VAD既定閾値 | VoiceActivityDetector.defaultThreshold | Float | 0.003 | VAD初期値 |
| VAD attack frames | attackFrames | Int | 1 | 発話開始判定 |
| VAD release frames | releaseFrames | Int | 40 | 発話終了判定 |
| pre-roll保持数 | preRollLimit | Int | 20 | 発話直前音声送出 |
| keepalive間隔 | keepaliveIntervalFrames | Int | 50 | 無音時接続維持 |
| mute自動停止遅延 | muteAutoStopDelayDefault | Duration | 2秒 | mute継続時の入力停止 |
| remote talker失効 | remoteTalkerTimeout | TimeInterval | 0.6秒 | 話者状態クリア |
| ジッタ再生遅延 | RemoteAudioJitterBufferDefaults.playoutDelay | TimeInterval | 0.015秒 | 受信平滑化 |
| ジッタ寿命 | RemoteAudioJitterBufferDefaults.packetLifetime | TimeInterval | 2.0秒 | 遅延パケット破棄 |
| 入力tap bufferSize | installTap bufferSize | Int | 128 | 低遅延入力取得 |
| 入力リサンプル先 | AudioResampler toRate | Double | 16,000Hz | 共通処理レート |
| 出力サンプルレート | SystemAudioOutputRenderer.sampleRate | Double | 16,000Hz | 再生レート |
| 出力チャンネル | SystemAudioOutputRenderer.channelCount | Int | 1ch | 再生チャンネル |
| HE-AAC処理レート | HEAACv2AudioEncoding.sampleRate | Double | 16,000Hz | codec処理レート |
| HE-AAC frame size | HEAACv2AudioEncoding.frameSize | Int | 2,048 samples | エンコード単位 |
| Audio Check録音時間 | startAudioCheck.recordDuration | Duration | 5秒 | 録音テスト長 |
| Audio Check再生時間 | startAudioCheck.playbackDuration | Duration | 5秒 | 再生テスト長 |

## 画面上の場所説明（ユーザー変更可能値）

| 画面 | セクション | 操作要素 | 変更できる値 |
|---|---|---|---|
| Settingsタブ | Audio I/O | Input Picker | 入力デバイス |
| Settingsタブ | Audio I/O | Output Picker | 出力デバイス |
| Settingsタブ | Transmit Codec | Codec Segmented Picker | PCM 16-bit / HE-AAC v2 VBR |
| Settingsタブ | Transmit Codec | HE-AAC v2 Quality Segmented Picker | Low / Medium / High |
| Settingsタブ | Audio Check | Voice Activity Detection Threshold Slider | VAD閾値 |
| Settingsタブ | Audio Check | Sound Isolation Toggle | Sound Isolation ON/OFF |
| Settingsタブ | Audio Check | Record 5s and Play Button | Audio Check実行 |
| Callタブ（selectedGroup選択後） | 通話ヘッダー Output | Output Slider | マスター出力音量 |
| Callタブ（selectedGroup選択後） | 通話ヘッダー Output | Mute/Unmute Output Button | マスター出力ミュート |
| Callタブ（selectedGroup選択後） | Participants 各行 | Output Slider | 参加者別出力音量 |
| Callタブ（selectedGroup選択後） | ローカルマイクヘッダー | Mute Toggle | ローカルマイクミュート |

## 通信仕様で使用する設定値

参照先: [通信仕様](通信仕様.md)

| 設定名 | 通信判定/通信送信での利用 |
|---|---|
| 送信codec | 利用する（送信payload codec選択） |
| HE-AAC品質 | 利用する（HE-AAC bitrate/品質） |
| VAD閾値 | 利用する（voice/keepalive生成タイミング） |
| ローカルマイクミュート | 利用する（音声送信停止 + peerMuteState送信） |
| 入力デバイス | 間接利用（送信元音声の入力ソースに影響） |
| 出力デバイス | 利用しない（再生経路のみ） |
| Sound Isolation | 間接利用（入力信号に影響） |
| Audio Check実行 | 利用しない（診断専用） |
| マスター出力音量 | 利用しない（受信後再生ゲインのみ） |
| マスター出力ミュート | 利用しない（受信後再生ゲインのみ） |
| 参加者別出力音量 | 利用しない（受信後再生ゲインのみ） |

## 保存と永続化

| 設定 | 永続化 |
|---|---|
| local member ID / display name | UserDefaultsへ保存 |
| 上記以外の音声系設定値（codec, VAD, I/O選択, 出力音量など） | プロセス生存期間のみ（再起動で初期値） |
