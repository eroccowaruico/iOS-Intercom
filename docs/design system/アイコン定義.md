# RideIntercom アイコン定義（SF Symbols）

本ドキュメントはデザインシステムの一部であり、SF Symbolの選定規約・レンダリング規約・一覧・是正定義を定める正本である。
本ドキュメントに違反するアイコン選定・レンダリング実装は不合格とする。

## SF Symbolレンダリング規約

- 必須: variableValue（可変値）対応シンボルは `Image(systemName:variableValue:)` を使用し、状態量を数値で渡す。固定表示で代替しない。
- 必須: マルチカラー対応シンボルは `.symbolRenderingMode(.multicolor)` を付与してマルチカラーで表示する。
- 禁止: マルチカラー対応シンボルに `.monochrome` を指定すること。
- 禁止: 可変シンボルの variableValue を省略し、常にフル表示（1.0固定）で済ませること。

## シンボルファミリー意味規約

- 必須: 下表のファミリーと用途の対応に従ってシンボルを選定する。
- 禁止: ファミリー規約に反する目的でシンボルを転用すること。

| ファミリー | 代表シンボル | 用途 | 禁止用途 |
|---|---|---|---|
| waveform系 | waveform, waveform.circle.fill, waveform.badge.mic, waveform.slash | 音声波形表示、音声処理（FX）、音声チェック、VAD状態 | マイク操作ボタン、レベル数値表示 |
| microphone系 | mic.fill, mic, mic.slash.fill | マイク本体の有効/無効/ミュート操作 | 音声波形表示、レベル表示、処理フロー状態 |
| microphone.and.signal.meter系 | microphone.and.signal.meter.fill | 入力レベル表示、音量可視化 | マイク操作ボタン、波形表示 |
| wifi系 | wifi, wifi.slash, wifi.exclamationmark | 接続状態（接続済み/接続中/オフライン）の表示 | 音声処理、送信チャネル表示 |
| antenna.radiowaves系 | antenna.radiowaves.left.and.right | 送信操作、通信チャネル状態 | 接続の良否状態表示 |
| network | network | トランスポート構成、通信経路のトポロジー表示 | 接続の良否状態表示 |

## アイコン是正定義

- 必須: 下表の旧シンボルを新シンボルに置き換える。
- 禁止: 是正後に旧シンボルを同用途で再使用すること。

| 対象 | 旧シンボル | 新シンボル | 理由 |
|---|---|---|---|
| VAD Silent / Waiting | waveform.path | waveform.slash | waveform.path は音声波形視覚化（FX等）へ転用する。VAD無音状態はゲート閉（波形遮断）として waveform.slash で表現する |
| VAD Voice（音声検出中） | waveform.path.ecg | waveform.badge.mic | 音声活動の検出/通過をマイクバッジ付き波形で表現する。波形視覚化ではなく検出/ゲート通過の意味にする |
| Input FX（非稼働/稼働共通） | slider.horizontal.3 | waveform（非稼働: neutral、稼働: success） | スライダー系は音量調整・設定操作の意味になる。音声FX処理には waveform ファミリーを使う |
| 接続状態（antenna.radiowaves を接続良否用途で使用している箇所） | antenna.radiowaves.left.and.right（接続良否文脈） | wifi（接続良否文脈は wifi ファミリーに統一） | antenna.radiowaves 系は送信チャネル専用とし、接続状態の良否（Connected/Connecting/Offline）は wifi ファミリーで統一する |

## 色種類と色の意味

| 色種類 | 色の意味 |
|---|---|
| default | 標準表示（文脈依存、明示的な色指定なし） |
| accentColor | 主要操作の強調 |
| textSecondary | 補助情報、通常状態の説明 |
| textTertiary | 弱い補助情報 |
| neutral | アイドル、未接続、基準状態 |
| info | 情報提示、受信中、完了表示 |
| success | 接続成功、再生中、正常処理中 |
| warning | 接続中、待機、注意 |
| danger | ミュート、削除、異常、遮断 |

## アイコン利用定義

- 必須: 下表に定義のないシンボルをアプリUIに追加する場合は、同時に本テーブルへ定義を追加する。
- 禁止: 下表の「禁止事項」に該当する用途でシンボルを使用すること。
- 必須: 同一シンボルを複数の状態で使う場合は、「色規則」の状態別定義をすべて満たす。
